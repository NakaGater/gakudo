ひまわり学童クラブは、練馬区立光が丘小学校区域の放課後児童クラブです。 小学1年生から6年生までのお子さまをお預かりし、安全で楽しい放課後の居場所を提供しています。 指導員6名体制で、宿題のサポートや季節の行事、外遊びなどを行っています。

対象 光が丘小学校に通う小学1年〜6年生

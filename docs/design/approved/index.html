施設について

地域の小学1年生から6年生までの児童を対象に、放課後の安全な居場所を提供しています。

QRコードで入退場管理
専用のQRコードで入室・退室をワンタッチ記録。保護者のスマホにリアルタイムで通知が届きます。
連絡事項をアプリで配信
行事予定やお知らせをアプリから配信。紙のプリントをなくし、既読確認も可能です。
活動写真の共有
施設での活動写真を保護者限定で安全に共有。行事の様子をいつでも振り返れます。
延長保育料金の自動計算
延長保育の料金を退室時間から自動計算。月次の請求書をオンラインで確認できます。

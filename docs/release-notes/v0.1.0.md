# リリースノート v0.1.0

## 新機能

### 公開ホームページ
施設の紹介、説明会の案内、お知らせを掲載できるホームページです。ログインしなくても誰でも閲覧できます。

### 入退場記録
お子さまのQRコードカードをパソコンのカメラにかざすだけで、入場・退場が記録されます。保護者の方はスマホでお子さまの登園状況をいつでも確認できます。

### 連絡事項
先生からの連絡事項をアプリで配信します。スマホでいつでも確認でき、過去の連絡事項も探しやすくなります。

### 写真共有
学童での活動写真を先生がアップロードし、保護者の方がスマホで閲覧できます。写真はログインした方のみ閲覧可能です。

### アカウント管理
先生が保護者のアカウントを作成し、お子さまと紐付けます。保護者の方は自分のお子さまの情報のみ閲覧できます。

## 既知の制限事項

- 動画の共有は次のバージョンで対応予定です
- 延長保育料金の自動計算には対応していません（入退場時間の記録は可能）
- 複数施設の管理には対応していません
- プッシュ通知には対応していません
- 連絡事項の既読管理には対応していません
